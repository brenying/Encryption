# Encryption Program

## Objective
This project implements a Java-based encryption and decryption tool that allows users to enter a custom cipher, encrypt plaintext into hexadecimal form, and decrypt previously encoded messages. It serves as a hands-on demonstration of input validation, ASCII manipulation, and fundamental cryptographic concepts.
